# ZWEI

**Zenith Workflow Executive Intelligence**

ZWEIは、メール・SNS・会議などの情報を解析し、タスク管理やスケジュール支援を行う**個人秘書AIシステム**です。

ユーザーのデジタル活動を常時監視し、必要な情報を整理・要約し、ToDo提案やリマインドを行います。

ZWEIは次の2つの側面を持つシステムです。

* **ワークフロー自動化システム**
* **人格を持つキャラクターAI**

AIの人格は、ゲーム *勝利の女神：NIKKE* のキャラクター「ツバイ」をモデルとしています。

---

# 概要

ZWEIは、ユーザーの生活や研究活動を補助する**常駐型AI秘書**です。

主な目的

* 情報の自動整理
* タスク抽出
* スケジュール管理
* 作業提案
* 行動支援

メール・SNS・会議などの入力情報を解析し、ユーザーにとって重要な情報を整理して通知します。

---

# 主な機能

## メール処理

受信したメールを自動解析し、以下を行います。

* メール要約
* 重要度判定
* タスク抽出
* 締切検出

通知例

```
ZWEI

先生、新しいメールを確認しました。

要約:
レポート提出依頼

締切:
3月20日

今日は少し進めておくと安心だと思います。
```

---

## カレンダー自動登録

メールや文章から

* 日付
* 時刻
* イベント

を検出し、カレンダーに登録します。

---

## ToDo提案

タスクと締切を元に、作業計画を提案します。

時間単位

* 今日
* 今週
* 今月

例

```
先生、レポートの締切が3日後です。
今日は30分ほど進めておくと安心だと思います。
```

---

## 会議議事録

会議の録音データから

* 議事録生成
* タスク抽出
* 日付検出

を行います。

---

## SNS監視

ユーザーのSNS投稿を監視し、必要に応じてコメントやリマインドを行います。

例

ユーザー投稿

```
今日は家二郎する
```

ZWEI

```
ラーメンもいいと思います。
ですが先生、レポートの締切も近いです。
```

# AI人格

ZWEIは単なるツールではなく、**人格を持つAI秘書**として設計されています。

性格

* 真面目
* 落ち着いている
* 丁寧な敬語
* 少し不器用だが気遣いがある

ユーザー呼称

```
先生
```

褒められた場合

```
...
ありがとうございます。
```

ユーザーに対して、**ごく控えめな好意**を持つ設定になっています。

# Docker構成

```
Docker
 ├ n8n
 ├ postgres
 └ zwei-worker
```

各コンポーネント

**n8n**

ワークフロー自動化

**postgres**

**zwei-worker**
